What is the DOM?

The DOM is short for Document Object Model which can be thought of as a big family tree. It holds all the information of a web document and is what links HTML, CSS and JavaScript together. You can access this tree of information on any webpage, whilst also being able to use it to manipulate elements.

When manipulating and changing elements on a webpage such as changing a heading, or a paragraph, this is only changing it for your version of that webpage. Once you hit refresh, the changes are gone and it will be back to its original state. However, for our own projects we can link our workspace to the DOM so that when changes are made, they are saved into the code. This saves us from having to only edit in our code editor and is very useful.
